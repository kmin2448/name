제목: 명패 자동 생성 웹 애플리리케이션 개발 요청
안녕하세요. 아래의 상세 사양을 충족하는 명패 제작 웹 앱을 React 또는 HTML/JS 기반으로 제작해 주세요. 
엑셀 명단을 업로드하면 설정된 디자인에 맞춰 출력용 PDF(또는 이미지 파일)를 생성하는 도구입니다.

1. 주요 기능 및 설정사이즈 선택: 

시중에 판매되는 표준 명패 사이즈 리스트를 드롭다운으로 제공하세요.
대(250mm x 90mm), 중(210mm x 70mm), 소(150mm x 60mm), 사용자 지정

배경 업로드 및 조정: 사용자가 배경 이미지를 업로드하면 선택한 명패 사이즈에 맞춰 배치합니다. 사이즈가 맞지 않을 경우 '채우기(Cover)' 방식으로 자동 확대/축소되도록 해주세요.

텍스트 레이아웃 설정: 화면에서 다음 항목의 폰트 크기, 굵기, 위치를 드래그나 수치로 조정할 수 있어야 합니다.

항목: 프로그램명, 소속, 이름, 직책데이터 연동: 엑셀(.xlsx, .csv) 파일을 업로드하면 각 행의 데이터를 바탕으로 명패가 대량 생성되어야 합니다.

2. 출력물 핵심 로직 (중요)양면 인쇄용 상하 반전: 한 장의 종이를 접어 양면 명패를 만들기 때문에, 동일 인물의 정보가 상하로 2세트 배치되어야 합니다.배치 방식: 위쪽 세트는 **180도 회전(상하 반전)**되어 있고, 아래쪽 세트는 정방향으로 배치하여 접었을 때 앞뒤에서 똑바로 보이도록 구성해 주세요.

3. 기술적 요구사항라이브러리: 엑셀 처리는 xlsx, PDF 생성은 html2canvas 및 jsPDF 라이브러리를 사용해 주세요.

프리뷰: 화면 우측이나 하단에 실제 데이터가 적용된 명패의 미리보기 화면을 실시간으로 보여주세요.

다운로드: 모든 명단이 포함된 다중 페이지 PDF 파일을 생성하는 버튼을 만들어 주세요.
